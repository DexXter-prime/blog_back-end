
## How to use

#### 1. Install all dependencies

```
  npm install
```

#### 2. Create tables

```
  npm run migrate-up
```


#### 3. Start the server

```
  npm run dev
```

#### To Destroy tables use

```
  npm run migrate-down
```







## Install before run

- [Download Node js](https://nodejs.org/en/)
- [Download PostgreSQL](https://www.postgresql.org/)
